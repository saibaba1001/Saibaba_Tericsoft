# Rick and Morty Explorer

I have created a premium React application using the Rick and Morty API to showcase a responsive UI with animations and glassmorphic design.

## Features
- Rick and Morty API integration
- Framer Motion animations
- Responsive grid layout
- Shimmering loading states
- Error handling with retry functionality

## Tech Stack
- React + TypeScript + Vite
- Framer Motion
- Lucide React Icons
- Vanilla CSS (Glassmorphism)
